Basic C Structure

Published: Mon 20 February 2023

In Blogs.

tags: tech c_programming

This tutorial is for basics of C Programming. It doesn't make you master or intermediate in this language. This tutorial will print Hello World on the terminal and explain the syntax of the basic program.

So, Let's Begin

First, Write the program in your favorite text editor and run it.

#include <stdio.h>

int main() {
    printf("Hello World");
    return 0;
}

output: Hello World

Let's disect the code blocks

#include<stdio.>

This line just simple import the standart I/O library. All the I/O operation are done throught this library.

int main()

In general, All function in C language must return a value. For example, this function return a integer value. We can change int in int main() to whichever value we want to return.

main() in int main() is the name of the function. main() is the starting point for C. When we run the program the compiler will look for main() and if unavailable it will give an error.

printf("Hello World");

printf() is the default output function in standart I/O library. We use this function and pass it a String to print on the terminal. Hello World is the string we passed to the function. Each statement in C must end with ";" in the end.

return 0

Simply returns a integer value.

Each functions in C must be put under the "{" and "}".

Let's wrap it up.
